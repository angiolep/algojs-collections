Class: Stack

algojs-collections/stack~ Stack

A stack implements a data structure which allows to store and retrieve objects in a last-in-first-out (LIFO) fashion.

foreach(fn)

The foreach method is meant to traverse all elements of the traversable, and apply the given operation fn to each element. The invocation of fn is done for its side effect only; in fact any function result of f is discarded by foreach.
